## Descripción

Sistema de Gestión de Talento Humano desarrollado para la Gobernación de Boyacá.

La aplicación permite administrar información relacionada con dependencias, servidores públicos y cargos, mostrando información consolidada en un panel principal para facilitar la consulta y gestión de datos.

## Tecnologías Utilizadas

- Java
- JavaFX
- PostgreSQL
- Gradle
- JPA (Persistence)

## Funcionalidades

- Gestión de Dependencias.
- Gestión de Servidores Públicos.
- Gestión de Cargos.
- Visualización de información mediante Dashboard.
- Conexión con base de datos PostgreSQL.
